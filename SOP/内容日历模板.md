## 内容矩阵

### 按内容类型分

| 类型 | 占比 | 目的 | 典型形式 |
|------|------|------|----------|
| 热点追踪 | 40% | 获取流量 | 短视频/快讯 |
| 深度分析 | 25% | 建立专业感 | 长文/中视频 |
| 实用干货 | 20% | 积累收藏 | 图文清单/教程 |
| 个人观点 | 15% | 塑造人设 | 短文/vlog |

### 按平台特性填充

利用 hot-creator 每日选题的输出，按上述矩阵分配：

1. 从 Dashboard 选出今日值得做的 2-3 个话题
2. 对照内容矩阵确定该话题属于哪个类型
3. 根据类型匹配最佳平台（参见[[平台策略指南]]）
4. 填入周度排期表

## 利用知识库辅助规划

```bash
# 查看持续热点（适合做系列内容）
python scripts/knowledge_base.py --stats

# 查看某个领域的历史热点
python scripts/knowledge_base.py --query "AI"

# 导出最近7天的趋势图谱
python scripts/export_mindmap.py -i output/briefs.json --cumulative --days 7
```
