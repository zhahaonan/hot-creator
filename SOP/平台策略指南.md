# 平台策略指南

## 平台特性速查

### 抖音

| 维度 | 要点 |
|------|------|
| 内容形式 | 竖屏短视频（15-60秒最佳） |
| 核心算法 | 完播率 > 互动率 > 关注量 |
| 最佳时间 | 工作日 12:00-13:00、18:00-21:00；周末 9:00-22:00 |
| 黄金时长 | 30-45秒（完播率最高区间） |
| 爆款要素 | 前3秒hook、情绪冲突、字幕清晰、BGM卡点 |
| 适合话题 | 民生财经、情绪事件、反常识、实用技巧 |

### 小红书

| 维度 | 要点 |
|------|------|
| 内容形式 | 图文笔记（6-9张图）/ 短视频 |
| 核心算法 | 收藏率 > 评论率 > 点赞率 |
| 最佳时间 | 工作日 7:00-9:00、12:00-14:00、20:00-23:00 |
| 封面要求 | 大字标题+副标题、高饱和色、信息密度高 |
| 爆款要素 | 干货感、清单化、避坑指南、亲身经历 |
| 适合话题 | 避坑指南、个人经历、干货整理、产品测评 |

### 知乎

| 维度 | 要点 |
|------|------|
| 内容形式 | 长文回答 / 文章 |
| 核心算法 | 点赞权重 > 收藏 > 评论 |
| 最佳时间 | 工作日 10:00-12:00、20:00-23:00 |
| 理想长度 | 2000-5000字 |
| 爆款要素 | 专业深度、数据支撑、逻辑清晰、反常识 |
| 适合话题 | 政策解读、技术分析、行业洞察、制度讨论 |

### B站

| 维度 | 要点 |
|------|------|
| 内容形式 | 中长视频（3-10分钟） |
| 核心算法 | 播放时长 > 投币 > 收藏 > 弹幕 |
| 最佳时间 | 周末 10:00-22:00；工作日 18:00-22:00 |
| 理想时长 | 5-8分钟 |
| 爆款要素 | 信息密度高、视觉丰富、章节清晰、互动弹幕引导 |
| 适合话题 | 科普解读、数据可视化、法律分析、深度复盘 |

### 微信公众号

| 维度 | 要点 |
|------|------|
| 内容形式 | 深度长文（3000-8000字） |
| 核心指标 | 在看 > 转发 > 阅读量 |
| 最佳时间 | 工作日 7:00-8:30、21:00-22:30 |
| 理想长度 | 3000-5000字 |
| 爆款要素 | 强标题、清晰结构、金句密度、引导转发语 |
| 适合话题 | 全景式报道、深度分析、行业趋势、商业解读 |

### 微博

| 维度 | 要点 |
|------|------|
| 内容形式 | 短文+图片 / 热搜话题博文 |
| 核心指标 | 转发 > 评论 > 点赞 |
| 最佳时间 | 全天（跟热搜节奏） |
| 爆款要素 | 时效性极强、站队明确、情绪引导、话题标签 |
| 适合话题 | 娱乐事件、社会热点、突发新闻 |

## 跨平台分发策略

### 核心原则

1. **先主后次**：选一个主打平台发完整版，其他平台发适配版
2. **错峰发布**：各平台间隔 2-4 小时，避免自我竞争
3. **形式适配**：同一内容不同平台必须调整形式

### 典型分发流程

```
时间轴:
  T+0h  抖音/微博 → 抢热度窗口（短平快版本）
  T+2h  小红书    → 精华图文版（信息图+要点）
  T+4h  知乎/B站  → 深度分析版（完整论据+数据）
  T+6h  公众号    → 全景长文版（最完整的内容）
```

### 平台优先级判断

| 话题类型 | 首发平台 | 原因 |
|----------|---------|------|
| 突发热点 | 微博/抖音 | 时效性最强 |
| 政策解读 | 知乎 | 用户需要深度 |
| 个人经历 | 小红书 | 真实感最强 |
| 科技产品 | B站 | 演示类内容优势 |
| 行业分析 | 公众号 | 决策者聚集 |
