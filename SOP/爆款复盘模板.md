# 爆款复盘模板

## 适用时机

- 单条内容数据明显高于平均值（2倍以上）
- 系列内容结束后的整体回顾
- 每周/每月固定复盘

## 复盘流程

### Step 1：数据收集（发布后 48-72 小时）

| 指标 | 数值 | 与均值对比 |
|------|------|-----------|
| 播放/阅读量 | | ×__倍 |
| 点赞数 | | ×__倍 |
| 评论数 | | ×__倍 |
| 收藏数 | | ×__倍 |
| 转发/分享 | | ×__倍 |
| 完播率（视频）| | __%  |
| 涨粉数 | | +__ |
| 流量来源 | 推荐__%/搜索__%/关注__% | |

### Step 2：归因分析

#### 选题维度

- [ ] 话题热度：发布时该话题处于什么阶段？（萌芽/上升/顶峰/下降）
- [ ] 时效性：是否在最佳窗口期内发布？
- [ ] 竞争度：同期有多少人做了同一话题？
- [ ] 差异化：我的角度和别人有什么不同？

#### 内容维度

- [ ] 标题/封面：点击率如何？A/B测试了吗？
- [ ] 开头3秒/首图：完播率/阅读完成率如何？
- [ ] 信息密度：内容是否提供了足够价值？
- [ ] 情绪触发：是否精准触发了目标情绪？
- [ ] CTA 效果：评论引导、转发引导是否奏效？

#### 分发维度

- [ ] 发布时间：是否在目标平台的黄金时间？
- [ ] 标签/话题：选择的标签是否精准？
- [ ] 跨平台：是否做了跨平台分发？效果如何？

### Step 4：知识库更新

将复盘发现录入项目，沉淀为可复用知识：

1. 成功模式 → 更新到 `config.yaml` 的 `content_domains` 关键词
2. 高效角度 → 记录到 `SOP/话题深度分析.md` 的角度选择表
3. 平台发现 → 更新到 `SOP/平台策略指南.md`
