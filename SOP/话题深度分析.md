# 话题深度分析 SOP

## 概述

对单个热点话题进行全方位深度拆解，产出可直接用于创作的完整方案。

## 适用场景

- 发现一个高潜力话题，想深入研究
- 某话题在知识库中持续多天出现，需要系列内容
- 竞品在做某话题，需要差异化角度

## 分析框架

### 1. 话题背景（5分钟）

- [ ] 事件起因和经过梳理
- [ ] 时间线整理：何时出现 → 何时爆发 → 当前阶段
- [ ] 利益相关方识别

**工具**：查看 Topic 笔记的「趋势概况」区块

### 2. 受众分析（5分钟）

- [ ] 核心受众画像（年龄/职业/兴趣）
- [ ] 受众关心的核心问题
- [ ] 受众情绪倾向（支持/反对/中立/焦虑）

**信号来源**：评论区热词、知乎回答、微博态度

### 3. 角度选择（10分钟）

| 角度类型 | 特点 | 适合平台 |
|----------|------|----------|
| 信息差 | 别人不知道的事实 | 知乎、B站 |
| 情绪共鸣 | 触动情感认同 | 抖音、小红书 |
| 反转/打脸 | 打破常见认知 | 全平台 |
| 实用指南 | 提供具体方法 | 小红书、公众号 |
| 观点输出 | 犀利独到见解 | 知乎、公众号 |

- [ ] 从 Topic 笔记的「创作角度」中选 1-2 个主攻角度
- [ ] 确认角度的差异化（搜索已有内容，避开同质化）
- [ ] 选定对应平台

### 4. 内容规划（10分钟）

- [ ] 确定内容形式（短视频/图文/长文）
- [ ] 撰写标题（从 Topic 笔记的「标题建议」中选或改编）
- [ ] 确定大纲结构（参考 Topic 笔记的「内容大纲」）
- [ ] 准备素材清单（数据/案例/图片/引用）

### 5. 跨平台适配（5分钟）

- [ ] 主打平台：完整版内容
- [ ] 次要平台：精华缩减版
- [ ] 各平台特有的格式要求（抖音竖屏、小红书封面、公众号头图）

## 输出模板

```markdown
# {话题名} - 创作方案

## 选定角度
- 角度：{角度名称}
- 理由：{为什么选这个角度}

## 主打平台
- 平台：{平台}
- 形式：{短视频/图文/长文}
- 发布时间：{具体时间}

## 标题
- 首选：{标题1}
- 备选：{标题2}

## 大纲
1. {开头/Hook}
2. {核心内容}
3. {高潮/反转}
4. {收尾/CTA}

## 素材清单
- [ ] {素材1}
- [ ] {素材2}
- [ ] {素材3}
```

## 关联知识库

分析完成后，可查询知识库中的相关历史：

```bash
python scripts/knowledge_base.py --query "{话题关键词}"
```
